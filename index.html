LA Pathways
LA Pathways is a SPA application developed for the City of Los Angeles Mayor's Office. The application connects new entrepreneurs and business owners to cultivate mentorship and coaching opportunities.

From Downtown
From Downtown is a personal full stack project I developed for my love of fantasy basketball. The idea behind the app is a user can input player names, and the app would retrieve stats for the selected players.

I set up a SQL database that stored basketball stats for individual players. I used NodeJs to create my own REST APIs. I used React and Bootstrap on the front end.
